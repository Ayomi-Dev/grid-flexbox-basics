# CSS Grid & Flexbox Basics

This repository contains a collection of layout examples and exercises I personally developed to deepen my understanding of **CSS Grid** and **Flexbox**. It serves both as a reference for myself and as a resource for anyone looking to learn modern CSS layout techniques in a clear and practical way.

## 📚 About the Project

The goal of this project is to demonstrate how CSS Grid and Flexbox can be used—individually and together—to create responsive, flexible, and visually appealing layouts. Each file showcases specific layout patterns and techniques, from simple one-dimensional Flexbox alignments to complex two-dimensional Grid structures.

This repo is ideal for:
- Beginners learning CSS layout systems.
- Developers refreshing their knowledge of Grid or Flexbox.
- Anyone looking for code samples that demonstrate layout solutions.

## 🚀 What You'll Find

The repository is organized into two main folders:

### 🔷 Flexbox
- Direction and wrapping examples
- Aligning items and content
- Building navigation bars, cards, and sidebars

### 🟩 Grid
- Explicit and implicit grids
- Grid areas and line placement
- Responsive grid templates
- Layouts like galleries and landing pages

### 🔀 Combined Use
Some layouts may show how both techniques can complement each other in a single design.
